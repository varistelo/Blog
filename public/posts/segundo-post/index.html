Introdução#

Vamos criar um ambiente de desenvolvimento local usando Docker Compose.

O docker-compose.yml#

version: '3.8'

services:
  app:
    build: .
    ports:
      - "3000:3000"
    volumes:
      - .:/app
    environment:
      - NODE_ENV=development

  db:
    image: postgres:15
    environment:
      POSTGRES_DB: myapp
      POSTGRES_USER: admin
      POSTGRES_PASSWORD: secret
    ports:
      - "5432:5432"
    volumes:
      - pgdata:/var/lib/postgresql/data

volumes:
  pgdata:

Executando#

docker-compose up -d
docker-compose logs -f app

Comandos Úteis#

# Parar tudo
docker-compose down

# Rebuild
docker-compose up --build

# Ver status
docker-compose ps

Conclusão#

Docker Compose simplifica muito o setup de desenvolvimento local.
